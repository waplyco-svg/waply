# Waply Site

Static marketing site for the zero-cost launch of Waply.

## Files
- `index.html`
- `services.html`
- `estimate.html`
- `styles.css`

## GitHub Pages Publishing
1. Create or use a GitHub account.
2. Create a public repository named `waply`.
3. Upload all files in this folder to that repository.
4. In repository settings, enable GitHub Pages from the `main` branch root.
5. The site will publish at:
   - `https://<github-username>.github.io/waply/`
